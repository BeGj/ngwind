What is ng-tailwind-ui?

A headless-first component set styled with Tailwind utility classes.

 How is accessibility handled?

@angular/aria provides the keyboard interactions, focus management, and ARIA roles.

How do styles reach my app?

Tailwind scans the library templates via the @source directive in your global CSS.
